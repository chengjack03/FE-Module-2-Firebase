# React E-Commerce App

A React + TypeScript e-commerce application with a full CI/CD pipeline using GitHub Actions and Vercel.

## Live App
[https://fe-module-2-firebase-lrap.vercel.app](https://fe-module-2-firebase-lrap.vercel.app)

## CI/CD Pipeline
- **Testing**: Jest + React Testing Library (unit & integration tests)
- **CI**: GitHub Actions runs tests on every push to `main`
- **CD**: Automatic deployment to Vercel when all tests pass

## Tech Stack
- React + TypeScript + Vite
- Redux Toolkit (cart state management)
- Firebase Authentication
- Jest + React Testing Library
- GitHub Actions
- Vercel
